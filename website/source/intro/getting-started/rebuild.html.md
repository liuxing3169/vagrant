# 重建

当你准备再次回到你的项目时，不管它是明天或者是一周后，或者是一年后，启动并运行它是非常简单的：

```
$ vagrant up
```

就这些！由于Vagrant环境已经全部配置到Vagrantfile中了，你或者你的同事在任何时间简单的运行`vagrant up` ，Vagrant将会自动重新创建你的工作环境。
